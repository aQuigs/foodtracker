# Decisions log

Append-only record of architectural decisions. Each decision gets its own file: `NNNN-short-name.md`.

Format is loose — for each decision capture:
- **Date**
- **Context** (what problem are we solving?)
- **Decision** (what did we choose?)
- **Alternatives considered**
- **Consequences** (what does this lock us into?)
